// 3- What are the tags and attributes in HTML? //In HTML, tags and attributes are used to define the structure and content of a web page. Tags HTML tags are used to define the different elements of a web page, such as headings, paragraphs, images, links, and more. Tags are surrounded by angle brackets (< and >) and usually come in pairs, with the opening tag preceding the content and the closing tag following the content. The closing tag is denoted by a forward slash (/) before the tag name. Here are some examples of HTML tags: h1, h2, h3, etc. for headings p for paragraphs img for images a for links div for divisions or sections span for inline elements Attributes HTML attributes are used to provide additional information about an element, such as its style, behavior, or relationship to other elements. Attributes are added to the opening tag of an element and consist of a name and a value, separated by an equals sign (=). Here are some examples of HTML attributes: src for specifying the source of an image or link href for specifying the hyperlink reference style for specifying the style of an element class for specifying the class of an element id for specifying the unique identifier of an element // 4- What are void elememts in HTML? // In HTML, void elements, also known as self-closing elements, are elements that do not have a closing tag. They are used to add a single piece of information to a document, and they do not wrap around any content. Characteristics of Void Elements: They do not have a closing tag. They cannot contain any content. They are typically used to add metadata or attributes to a document. Examples of Void Elements: img: used to add an image to a document. br: used to add a line break to a document. hr: used to add a horizontal rule to a document. link: used to add a link to an external stylesheet or script. meta: used to add metadata to a document, such as the character encoding or author information. // 5- What is the feature of HTML ? //HTML (Hypertext Markup Language) is a standard markup language used to create web pages.
